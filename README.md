# 🚀 Deskspace Booking Website

A modern and responsive coworking space landing page that allows users to explore shared office spaces, search locations, and view workspace features. Built with Bootstrap and optimized for clean UI/UX.

---

## ✨ Features

- 📍 Location-based workspace search
- 📅 Date selection for booking
- 🏢 Popular locations showcase (Dubai, New York, etc.)
- 📊 Statistics section (available seats, workspace data)
- 💼 Workspace features (Wi-Fi, Lunch/Dining, Dedicated Desk)
- 🎯 Clean and modern UI/UX design
- ❓ FAQ section with accordion
- 📩 Newsletter subscription form
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Fast and lightweight design using Bootstrap
